  ### Book Club サークルのための読みたい本リスト機能を持つアプリ
  
  Python 3.10.10
  pipenv 
  https://pipenv-ja.readthedocs.io/ja/translate-ja/basics.html
  
  Flask 
  firebase Firestore database, Authentication
  
  #### 開発
  ```
  仮想環境を起動
  pipenv shell
  
  仮想環境を起動停止
  pipenv exit 
  
  仮想環境内にてライブラリのインストール
  pipenv install [ライブラリ名]
  
  仮想環境内にてサーバー起動
  flask run 
  
  サーバーの起動停止
  ctrl + C 
  
  ヘロクにログインする
  heroku login
  
  デプロイ
  git heroku push main 
  
  デプロイしたページを開く
  heroku open
  ```
